Проект CRM для строительной компании

Этот проект реализует мини-систему CRM для строительной компании, позволяющую управлять квартирами и продажами через менеджеров. Включает функции регистрации менеджеров, управления клиентами и отслеживания статуса квартир.

Развернуто на Render для просмотра: https://construction-crm.onrender.com
